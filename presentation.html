Slide 5 // Backend Database

Google Sheets Backend API

Serverless Database Architecture

Form submissions are converted to URL-encoded datasets to bypass browser preflight CORS blockages. They are processed securely by a custom-deployed Google Apps Script, which inserts the entries into a Google Sheet in real-time.

Includes a local storage fallback system if offline or URL is missing.

// Serialization Pipeline
const formData = new URLSearchParams();
for (const key in data) { formData.append(key, data[key]); }

Slide 7 // Technical Evaluation

Advantages & Disadvantages

thumb_up Key Advantages

  • Zero Database Overhead: Google Sheets provides a free backend database.
  • Premium Interactivity: Features comparison grading slider and timestamp timeline jump selectors.
  • Offline-Ready fallback: Gracefully falls back to browser localStorage.

warning Key Limitations

  • Client-Side Only Session: Real-time feedback requires browser sessions to stay local unless synchronized with an external server database.
  • Apps Script Redirect Block: Apps Script redirection headers are blocked by browser CORS, requiring custom response-bypassing wrappers.
